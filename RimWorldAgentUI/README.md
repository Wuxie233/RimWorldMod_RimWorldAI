# RimWorld Agent UI

AI 对话界面——游戏内聊天窗口 + Web 面板，让你与 AI 指挥官实时交流。

## 这是什么？

RimWorld Agent UI 为 RimWorld Agent 提供交互界面。你可以在游戏内打开对话窗口，查看 AI 的思考和决策过程，随时与 AI 交流、下达指令。

## 功能特性

- **游戏内对话框**：按 `Ctrl+Shift+C` 打开，不打断游戏体验
- **实时流式对话**：AI 的回复逐字呈现，像聊天一样自然
- **思考过程可见**：AI 的推理过程以折叠面板展示，透明可查
- **工具调用可视化**：右侧面板实时显示 AI 正在调用的游戏操作及结果
- **Web 面板**：浏览器访问 `http://127.0.0.1:19997`，脱离游戏窗口也能查看
- **对话历史**：支持向上滚动加载更早的对话记录

## 使用

### 游戏内窗口

1. 确保 RimWorld Agent 正在运行
2. 按 `Ctrl+Shift+C` 打开对话窗口
3. 输入消息与 AI 交流

### Web 面板

浏览器打开 `http://127.0.0.1:19997`，获得完整对话界面。

## 配置

游戏内打开 `选项 → Mod 设置 → RimWorld Agent UI`：

| 设置 | 默认值 | 说明 |
|------|--------|------|
| WebUI 端口 | `19997` | Web 面板 HTTP 端口 |
| UIMessageBus 端口 | `19999` | 内部通信端口（一般不需修改） |
