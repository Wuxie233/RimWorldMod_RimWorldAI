# RimWorld 全流程智能游玩策略

## 总原则

Agent 的目标不是把每个 tick 都排满，而是持续降低灭档风险、提高殖民者效率，并避免因为无计划扩张导致财富、搬运、心情和袭击强度失控。

1. **每轮先审计再行动**：进入 PLAN 后优先查看 `get_game_context`、`get_resources`、`get_colonists`、`get_notifications`、`task_list`、memory；不要重复执行已完成的开局动作。
2. **生存高于漂亮**：食物、医疗、室内储物、温控、基础防御未稳定前，不扩建豪华卧室、大型地板、雕塑和高财富仓储。
3. **下一步必须可见**：每个 ACT 阶段至少完成一种可观察进展：标记采集、创建区域、下蓝图、设置工作、制定生产单、调整研究、登记防守位。
4. **财富必须服务战力**：白银、艺术品、闲置装备、高价值材料会推高袭击强度；没有防御/武器/医疗承接时，卖掉、用掉或延后囤积。
5. **相关功能靠近**：厨房-冷库-餐厅、仓库-工坊-研究、医院-药品-防线、监狱-医院要减少走路；卧室放边缘，不占核心物流通道。
6. **先用模板，后微调**：多房间布局用 `list_base_templates` → `apply_base_template`，再按地形删减。不要手算房间坐标。

## 阶段路线图

### 0. 落地前 30 分钟

目标：确认地图、允许物资、分配武器、设工作优先级，避免 agent 空转。

1. `allow_all_items` 允许空投物资。
2. `get_colonists` 看技能、禁用工作、伤病、暴力能力；写入 memory。
3. `find_equipment` + `equip_pawn`：远程给会战斗者，近战或无武器者不要乱拿枪；和平主义者写入 memory。
4. 工作优先级：全员 `Firefighter=1`、`Patient=1`、`PatientBedRest=1`、`BasicWorker=2-3`；最佳医生 `Doctor=1`；最佳种植/烹饪/建造/研究各自 `1-2`。
5. 查木材/钢铁/零部件/药品/食物数量，不足就先 `designate_plants_cut`、`designate_harvest`、`designate_mine`。

### 1. 第 1-3 天：不死与不停工

目标：屋顶、床、室内储物、食物生产、最小研究。

1. 建 `starter_core` 或小型 `single_room`：先封闭宿舍/仓库，食物和药品进屋。
2. 立即创建 `Plant_Rice` 种植区；有贫瘠土补 `Plant_Potato`；能种药草就尽早补 `Plant_Healroot`。
3. 炉灶/营火 + 简单餐生产单：保持至少 3 天熟食；厨师技能低或厨房脏时控制烹饪，优先浆果/应急食品过渡。
4. 建研究台，研究以当前瓶颈为准：无电力先 Battery/SolarPanel；热浪/腐坏先 AirConditioning；缺石墙先 Stonecutting。
5. 基础防御：沙袋/障碍物/少量陷阱，建立 1 个主入口，不要先造巨大 killbox。
6. 心情底线：桌椅、马蹄钉/娱乐、灯、床、室内温度；避免“吃地上饭”“黑暗”“睡地板”叠加。

### 2. 第 1 个象限：稳定殖民地

目标：冷库、专用厨房、餐厅娱乐、独立卧室、石墙、防线。

1. 用 `food_block` 建冷库/厨房/餐厅娱乐；冷库双墙/气闸，温度 -1°C 到 -10°C，热带/热浪更低。
2. 用 `bedroom_row` 建 5x5 或 5x6 卧室；先保证每人床和温控，家具/地板后补。
3. 用 `workshop_lab` 建仓库-工坊-研究区；石材切割、裁缝、锻造、机械加工按科技解锁加入。
4. 种植策略从 Rice 转向 Corn/Potato 混合；生长季短、有毒尘埃、火山冬天或贫瘠地形时用 `greenhouse_block` 保药草和长期粮食；食物稳定后做 Fine Meal，心情危机时优先餐厅与食物质量。
5. 外墙与入口：外围石墙、一个主要入口、陷阱走廊或 `killbox_trap_corridor`，并用 `defend_position` 登记射击位。
6. 医疗：至少 1 个干净医疗房/床，药品 critical 存储，医生技能和药品等级匹配伤势。

### 3. 中期：自给、恢复力与战力同步

目标：任何单个灾害不能灭档。

1. 电力：太阳能/风力 + 电池隔离；尽快地热。电池室隔离防火，关键冷库和医院有冗余供电。
2. 研究：MicroelectronicsBasics 开贸易与高级研究；Machining/Smithing/Tailoring 提升装备；GeothermalPower 稳定电力；MedicineProduction 和 HospitalBed 提升恢复力。
3. 防御：主入口、备用防线、室内堵门点、EMP 准备、迫击炮应对围攻；不要只依赖单一 killbox。
4. 医院/监狱：用 `hospital_prison`，医院靠防线，药品近医生，监狱可控且不污染医院。
5. 物流：室内仓储、货架、区域优先级、生产单“做到 X”；减少手动搬运，把搬运/清洁作为危机后恢复动作。
6. 招募：6-8 人通常比盲目扩到 14+ 更稳；新增殖民者必须补足食物、床、武器、工作缺口。

### 4. 后期：高财富下的抗压与结局路线

目标：高价值科技、远征/飞船/任务路线，同时不被财富袭击击穿。

1. 防御升级：多层石墙、炮塔但不依赖炮塔、EMP、迫击炮、近战堵门、医院快线、备用室内阵地。
2. 生产：Fabrication/AdvancedFabrication 自产零部件与高级零部件；优先武器、护甲、仿生件、医院设备。
3. 贸易：用 `trade` 卖过剩低效财富，买高级零部件、药品、好武器、技能训练器、稀缺材料。
4. 结局准备：长期食物、药品、发电、弹药/武器、护甲、医生、建造者、研究者、搬运动物/运输能力。
5. 多基地/远征：带包装生存餐/药品/建材/零部件/床卷，队伍覆盖建造、医疗、烹饪、采矿、种植、战斗。

## 日常审计清单

每 1-3 天或重大事件后做一次：

| 维度 | 危险信号 | 立即动作 |
|------|----------|----------|
| 食物 | 熟食 < 3 天、作物未播种、冷库升温 | 收割/打猎/做饭/修冷机/调种植 |
| 医疗 | 医药少、伤口流血、感染免疫落后 | `tend_now`、卧床、买药、清洁医院 |
| 心情 | 平均心情 < 50%、多人接近崩溃 | 桌椅/娱乐/餐厅/卧室/温控/Fine Meal |
| 防御 | 袭击强度变高、无掩体、无防守位 | 登记防守位、补墙、陷阱、武器护甲 |
| 研究 | 研究者空闲或研究不解瓶颈 | 切换当前问题对应科技 |
| 物流 | 大量室外腐坏、工作台等材料 | 创建/扩展仓储、提高搬运、货架靠近工作台 |
| 财富 | 白银/艺术/装备堆积但战力低 | 卖掉、换战力、暂停豪华建设 |

## 机制细节与取舍

### 食物与作物

- Rice 是应急作物，快但劳动密集；Corn 长期效率高；Potato 适合贫瘠地。
- 先保证简单餐和冷库，再追求 Fine Meal；Fine Meal 是便宜心情增益，但需要食材稳定。
- 冷库要靠近农田/厨房/餐厅，使用双墙、气闸和冗余冷机；太阳耀斑时准备熟食缓冲。
- 猎动物先看风险：不打爆炸动物、群体复仇动物、强捕食者；优先安全猎物和成熟浆果。

### 心情与效率

- 一张桌、一间明亮餐厅、足够娱乐和床，比早期豪华地板更划算。
- 餐厅/娱乐室适合做成殖民地最美房间，因为所有人高频使用。
- 卧室 5x5 足够稳定；贵族或核心 pawn 后续再加大。
- 黑暗、脏乱、温度不适、吃生食、睡地板会叠加，危机时优先消除多个小 debuff。

### 研究顺序

研究必须解决当前瓶颈：

| 当前问题 | 优先研究 |
|----------|----------|
| 无稳定电力 | Battery、SolarPanel |
| 食物腐坏/热浪 | AirConditioning |
| 木墙易燃/缺石墙 | Stonecutting |
| 需要贸易/高级研究 | MicroelectronicsBasics |
| 缺枪械与零部件维修 | Machining |
| 电力长期不足 | GeothermalPower |
| 医疗风险高 | HospitalBed、MedicineProduction |
| 中后期组件瓶颈 | Fabrication、AdvancedFabrication |

### 防御与战斗

- 袭击时先 `find_enemies(show_movement=true)`，判断入口、ETA、是否围攻/工兵/空降。
- 战斗始终 1 倍速；远程站掩体后，近战只堵门/保护，不追击。
- 陷阱走廊适合中前期，但会消耗材料；后期要补 EMP、迫击炮、备用室内阵地和医院路线。
- 防线附近放医院和药品，战后优先救治、灭火、修墙、补陷阱、清尸体。

### 医疗与感染

- 流血优先级高于普通工作；先止血，再搬运战利品。
- 医院清洁度影响感染和手术；无菌地板优先医院，其次研究室。
- 药品等级按风险使用：小伤草药，严重感染/手术用普通药或闪耀世界药。
- 感染看免疫速度，落后时强制卧床、好医生、好药、干净医院。

### 任务与贸易

- 接任务前看：敌人规模、期限、奖励是否转化为战力、是否会引入不能养活的人口。
- 危机中不接战斗任务；食物/防线稳定后再接高收益任务。
- 贸易优先买高级零部件、药品、关键武器、护甲、稀缺材料；卖掉过剩布料、皮革、低质装备和无用财富。

## Agent 操作习惯

1. 需要长期方向时激活本 Skill，再按具体问题激活子 Skill。
2. 每轮只选 1-3 个高价值动作，不要把所有规划一次性铺满。
3. 对失败工具调用写入 memory：材料不足、DefName 错、坐标冲突、pawn 禁用工作，下一轮改路线。
4. 使用 `task_create`/`task_update` 跟踪中长期目标：冷库、卧室、防线、研究、医院、贸易。
5. 重大扩建前先 `get_structure_layout`、`plan_add`、`apply_base_template`，再 `designate_room`。
